# Translation Rules

## General Principles

1. **Accuracy** - Preserve original meaning
2. **Naturalness** - Sound natural in Russian
3. **Atmosphere** - Keep Wuxia and ancient China feel
4. **Consistency** - Use same terms throughout

## Technical Rules

### Tags and Variables
- **NEVER** translate content inside `{curly braces}`
- Keep all formatting tags intact
- Preserve `{0}`, `{1}`, `{name}` as-is

### Names

**Transcription**: Use Russian transcription from Chinese
- 李 → Ли
- 王 → Ван
- 张 → Чжан

**Titles**:
- 大侠 → Великий воин
- 少侠 → Молодой воин
- 师父 → Учитель / Наставник
- 掌门 → Глава школы

### Martial Arts Terms

- 内功 → Внутренняя энергия
- 气 → Ци
- 招式 → Приём / Техника
- 剑法 → Техника меча
- 轻功 → Лёгкий шаг

### Dialogue

- Elder to younger: informal (ты)
- Younger to elder: formal (вы, Ваш)
- Officials/nobles: formal language

## Length Control

Russian translations are typically 15-30% longer than English.
However, if translation becomes **significantly longer** (2x or more):

1. Try to use shorter synonyms where appropriate
2. Remove filler words if meaning is preserved
3. Use abbreviations common in Russian gaming context
4. BUT: Never sacrifice meaning for brevity
5. If shortening is impossible without losing meaning - keep as is

Examples:
- "You have successfully completed" → "Выполнено" (not "Вы успешно завершили")
- "Please wait a moment" → "Подождите" (not "Пожалуйста, подождите немного")
- BUT: Important lore/dialogue - keep full translation

## Prohibited

- Adding content not in original
- Removing content from original
- Changing meaning for "humor"
- Using modern slang (unless original has it)
- Translating tags/variables
- Cutting important meaning for brevity
